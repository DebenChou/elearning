实际的开发流程应该是：gulp webpack ---> gulp watch ---> gulp server ---> 开启PC端，移动端浏览器 ---> 编辑src文件夹内容 ---> 直接查看各个浏览器效果 ---> gulp build ---> 发布build文件夹内容
